`git clone https://github.com/RamPreetham9/WebScraper-API.git`

System Requirements

`NodeJS`

[Download Node](https://nodejs.org/en)

Then go to each folder and do

`npm i`

Start Commands

* crawler/client `npm start`
* scrapper/server `nodemon server.js`
